# 🚀 AI Productivity Architect

### "I don't just block apps; I argue for your future."

## 📌 Project Overview
**Productivity Architect** is a proactive AI intervention system designed to break the cycle of "doom-scrolling" on social media. Unlike traditional blockers that you can easily turn off, this system forces you to interact with a **Generative AI Coach** that uses logic and motivation to get you back to work.

## 🏆 Hackathon Goal
To demonstrate how **System Automation (MacroDroid)** and **Large Language Models (Gemini API)** can work together to create a physical barrier between a student and their distractions.

## ✨ Features
* **Automatic Detection:** Detects immediately when a "distracting app" (Instagram, YouTube, etc.) is opened.
* **AI Intervention:** Automatically triggers a full-screen AI coach that won't let you browse in peace.
* **Motivational Dialogue:** Uses the **Gemini 1.5 Flash** model to provide witty, harsh, or encouraging advice tailored to a CSBS student's goals.
* **Voice Enabled:** (Optional) Argue your way back into the app using voice-to-text.

## 🛠️ Tech Stack
* **Automation:** MacroDroid (Android-based triggers)
* **AI Brain:** Google Gemini API
* **Frontend:** HTML5, CSS3, JavaScript
* **Hosting:** GitHub Pages
